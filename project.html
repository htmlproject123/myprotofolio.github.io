Project Name:

Clustering Based Speech Emotion Recognition By Incorporate Learned Features And DeepLSTM

Description:

In Speech Emotion Recognition project, we propose a novel approach for speech emotion recognition by learned features and deep long short-term memory (LSTM) networks in a clustering-based framework. The recognition of a speaker's emotional state is a difficult task for machine learning algorithms, which are important in the field of speech emotion recognition (SER). SER is used to analyse the emotional state of speakers in many real-time applications such as human behaviour assessment, human-robot interaction, virtual reality, and emergency centres.
